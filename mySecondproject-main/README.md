# Garry Sehmbey Portfolio

Static portfolio website built with HTML, CSS, and JavaScript.

## Preview

Open `index.html` in your browser.

## Customize

Replace these placeholders before publishing:

- Project names, descriptions, and screenshots
- About section text
- Domain/brand wording if you want the domain shown on the page

## Deploy With Your Domain

Simple hosting options:

- GitHub Pages
- Netlify
- Vercel
- Your domain provider's static hosting, if available

After hosting, point your domain DNS to the hosting provider using their instructions.
